# Maegan Clark — Personal Brand Website

A single-file personal brand website built in HTML/CSS/JS.

## Files

- `maegan-clark.html` — The complete website (all styles, scripts, and images embedded)

## Pages & Sections

- **Home / Hero** — Intro, tagline, CTA
- **About** — Bio, headshot, tags
- **Work** — Cards linking to Rare Gem Media, Southern Blondie, LinkedIn, Pro Bono
- **Pro Bono** — Little Rock: You're Open feature
- **Connect** — Email CTA (hey.maeganclark@gmail.com)
- **Links** — Standalone page (toggled via nav) with Amazon Storefront and ShopMy cards
- **Footer** — Social links (Instagram, TikTok, Pinterest, LinkedIn)

## Deployment (GitHub + Vercel)

1. Create a new GitHub repository (e.g. `maeganclark-site`)
2. Upload `maegan-clark.html` and rename it to `index.html`
3. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
4. Vercel will auto-detect and deploy — no build settings needed
5. Add your custom domain under Project Settings → Domains

## Before Going Live

- [ ] Update the canonical URL in the `<head>` from `https://maeganclark.com` to your actual domain
- [ ] Update the Open Graph `og:url` to match your actual domain
- [ ] Add an `og:image` tag pointing to a hosted headshot for rich link previews on social
- [ ] Submit your URL to [Google Search Console](https://search.google.com/search-console) for indexing
